# OOPS Concepts

### This project demonstrates the use of Object-Oriented Programming (OOP) concepts in Java.

## Concepts Covered:
1. **Classes and Objects**: Creation of classes and instantiation of objects.
2. **Encapsulation**: Use of private fields and public getter/setter methods.
3. UML Diagrams: Visual representation of class structures and relationships.
